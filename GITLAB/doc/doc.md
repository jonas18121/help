## Versionner une application

Exemple :

**Symfony v5.4.2**

**vX.Y.Z**

**v** : Version

**X** : C'est grosse maj

**Y** : C'est modification avec breaking change ou ajout d'une feature importante

**Z** : C'est corrections mineures ou petits ajouts
