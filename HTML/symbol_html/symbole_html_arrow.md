# Symbole html Arrows

Site [toptal](https://www.toptal.com/designers/htmlarrows/arrows/)



## LEFT ARROW ←

- U+02190 : UNICODE, 

- `&#x2190;` : HEX CODE, 

- `&#8592;` : HTML CODE, 

- `&larr;` : HTML ENTITY, 

- \2190 

```html
<p><span style="font-size:100px;">&#8592;</span></p>
```
